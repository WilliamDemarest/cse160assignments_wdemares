Notes to grader:

- To display the graphic for the first extra task, use the "Draw Picture" button. The sketch I used to plan
it is in the Assignment_1 folder, outside src or lib. When I was planning it I was thinking more like a gl strip.
When I made it I used regular triangles instead, but I used the same pattern/points I had drawn. I also decided to
give the shapes 20 sides insted of the 10 I drew, which was a simple change using my Ring class.

- For the second, open ended extra task I made a python script that takes a .png image file
and generates a javascript file with instructions to recreate the image using the paint tool classes.
I used this to make instructions for two images, which can be displayed with the "Machine Picture A"
and "Machine Picture B" buttons. One uses the point class and so looks pixelated, and the other uses
the circle class to give a dot painting effect. The instruction files, python script, and source images
are in the lib folder.

Please use a browser that supports "canvas"
